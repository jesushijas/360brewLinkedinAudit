# 360brewLinkedinAudit
This repo contains a Claude Skill that audits user Linkedin profile and activity from 360Brew perspective (new Linkedin algorithm, 2026), scores it and provides a recommended plan based on the analysis

Cómo instalar la skill 360Brew Audit en Claude Cowork:

Descarga Claude para escritorio → claude.ai/download
Instala la extensión "Claude in Chrome" desde la Chrome Web Store
Descarga esta carpeta completa (Code → Download ZIP)
Descomprime el ZIP
Copia la carpeta 360brew-audit en tu directorio de skills de Claude:

Windows: C:\Users\TU_USUARIO\Claude\.claude\skills\
Mac: /Users/TU_USUARIO/Claude/.claude/skills/


Abre Claude, abre tu perfil de LinkedIn en Chrome, y escribe: "Evalúa mi perfil de LinkedIn"
